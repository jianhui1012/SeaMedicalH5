# SeaMedicalH5
####  海上医疗移动客户端实现了地图定位,事故上报,天气查询,事故受伤部位统计,海上职业病查询等功能!

## 项目
### 核心技术与框架
> 移动客户端采用基于H5+的MUI前端框架

> 后台采用EF为基础的ORM框架

> 接口采用使用Restful风格的WebApi
